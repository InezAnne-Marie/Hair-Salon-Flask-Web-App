API Access

Access our services data through our REST API:

  • GET /api/v1/resources/services/all: Get all services
  • GET /api/v1/resources/services?id=X: Get service by ID
  • GET /api/v1/resources/prices/all: Get all prices
  • GET /api/v1/resources/serv_prices?id=X: Get price by service ID
  • GET /api/v1/resources/expensiveservice: Get most expensive service
  • GET /api/v1/resources/cheapestservice: Get least expensive service
  • GET /api/v1/resources/services/sort: Get services sorted by name
  • GET /api/v1/resources/prices/sort: Get services sorted by price
